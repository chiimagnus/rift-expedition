## B. 首章关键路线

推荐首次使用战士 + 法师，从新游戏开始完整执行：

`village_square` → `village_outskirts` → `wilds_road` → `wilds_ruins` → `cave_entrance` → `cave_mines` → `cave_depths` → 返回 `village_square`

同时走完河岸支路：

`village_square` → `village_riverside` → `wilds_riverbank` → `cave_entrance`

- [ ] 九个区域全部进入至少一次。
- [ ] 每个出口完成一次双向往返。
- [ ] 切图后出生点不在障碍、NPC、道具或遭遇区内。
- [ ] 已胜利遭遇在切图和读档后不再触发。
- [ ] 战败遭遇仍可重新挑战。
- [ ] 缺少遭遇定义时触发器不会被永久消耗。

## C. 任务与内容闭环

### 主线：血债

- [ ] 与村长对话后可接受 `blood_debt`。
- [ ] 元素矿账册可在元素矿道获得。
- [ ] 返回村长时，缺少账册不能完成任务。
- [ ] 持有账册时可完成任务并获得裂隙碎片护符与水球术。
- [ ] 完成后进入章节结算，任务不会重复发奖。

### 支线：苦根草药

- [ ] 与药师接受 `bitterroot_medicine`。
- [ ] 苦根草可在苦根河湾获得。
- [ ] 交付后草药被移除，奖励正确加入背包。

### 支线：焦痕誓言

- [ ] 与梁铮接受 `scorched_vow`。
- [ ] 焦痕戒指可在旧驿站废墟获得。
- [ ] 交付后戒指被移除，奖励正确加入背包。

### 支线：矿工最后一班

- [ ] 与守卫阿砾接受 `miners_last_shift`。
- [ ] 矿工铁护手可在元素矿道获得。
- [ ] 交付后护手被移除，奖励正确加入背包。

### 状态一致性

- [ ] 任务接受、进行、完成状态在保存/读取后保持一致。
- [ ] 任务物品拾取后不会因切图或读档重复生成。
- [ ] NPC 对话会根据任务状态切换到正确脚本。
- [ ] 四个任务均不能在缺少交付物时完成。

## D. 地图、碰撞与遮挡

每个区域都执行：沿边缘绕行一圈、点击障碍另一侧、穿过最窄通路、在出口附近反复进入离开，并按 `D` 对照调试叠层。

- [ ] `village_square`：房屋、井、屋顶遮挡和两条出口一致。
- [ ] `village_riverside`：水域、堤岸、树冠和河湾出口一致。
- [ ] `village_outskirts`：篱笆、树林和主路开口一致。
- [ ] `wilds_road`：弯路、树丛和两处分岔出口一致。
- [ ] `wilds_ruins`：废墟墙体、油地表和入口一致。
- [ ] `wilds_riverbank`：浅水、深水、苦根草和洞穴侧路一致。
- [ ] `cave_entrance`：岩壁、洞顶、毒/火地表和矿道入口一致。
- [ ] `cave_mines`：矿车、岩壁、任务物品和出口一致。
- [ ] `cave_depths`：洞顶、裂隙区域、最终遭遇和返回路径一致。
- [ ] 角色走到前景后方时被正确遮挡，离开后立即恢复。
- [ ] 队员、NPC 与敌人没有明显的 Y 排序颠倒。

## E. 战斗机制与反馈

第二轮建议弓手 + 盗贼，覆盖远程攻击和高机动行为。

- [ ] 移动、普通攻击、技能、物品和结束回合均可执行。
- [ ] AP 不足、超出射程和视线受阻均给出清晰拒绝反馈。
- [ ] 近战命中只产生一次冲击反馈。
- [ ] 远程、奥术、火焰和毒素效果从施法者移动到目标。
- [ ] 伤害浮字等于实际扣除值。
- [ ] 治疗浮字等于实际恢复值，且不使用伤害表现。
- [ ] 闪避显示“闪避”，不播放命中冲击或受击动画。
- [ ] 致死攻击后目标呈明确倒地静态姿态，不继续待机。
- [ ] 火、水、油和毒地表的颜色及结果可以区分。
- [ ] 战斗胜利、全队失败和失败后恢复流程不会卡住。

## F. 音频状态与混音

- [ ] 村庄、河岸、荒野和洞穴使用正确的音乐及环境循环。
- [ ] 探索进入战斗时新音乐淡入、旧音乐淡出，无突兀截断。
- [ ] 战斗结束后恢复当前区域探索声景。
- [ ] 快速连续触发探索 → 战斗 → 探索不会误停新循环。
- [ ] 章节结束和回主菜单后所有声景停止。
- [ ] 多次往返区域不会造成环境声或音乐叠加变响。
- [ ] 主音量、音乐、环境声、音效和静音分别即时生效。
- [ ] 缺失单个音频资源时游戏继续运行，不崩溃。
