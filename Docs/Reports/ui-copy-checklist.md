# UI 文案与无障碍检查清单

检查日期：2026-07-06

## 已检查界面

- 主菜单：按钮、状态文案、章节引导均为中文。
- 职业选择：职业卡、属性、初始技能说明、开始/返回按钮均为中文；职业卡有读屏标签。
- 探索 HUD：移除了临时“和村长交谈”按钮；改为地图点击、互动、Tab 切队长提示。
- 地图视图：SpriteKit 地图层使用统一世界坐标，支持窗口缩放后的居中显示；地图视图有基础无障碍说明。
- 战斗 HUD：行动顺序、技能、消耗品、结束回合、返回按钮均为中文；`AP` 保留为规则术语，生命值不再显示英文 `HP`。
- 对话：对话选项来自中文数据；选项按钮有读屏标签。
- 背包与角色：共享背包、装备、属性加点均为中文；角色、装备、加点按钮有读屏标签。
- 任务日志：标题、目标、状态均为中文；任务条目合并为可读描述。
- 存档：5 个手动槽与 5 个自动槽说明、保存/读取按钮均为中文；槽位操作有读屏标签。
- 设置：音频、界面缩放、键位说明、调试叠层均为中文；滑块和开关有读屏标签。
- 调试叠层：标题已改为“调试”；该叠层只在用户主动开启时显示。
- 章节完成：总结、读取存档、返回主菜单均为中文；关键按钮有读屏标签。

## 允许保留的非中文术语

- `AP`：已在需求中作为整数行动点术语确认，战斗与技能文案保留。
- `Tab`、`D`：键盘按键名，设置页保留原始键名。
- 内部资源 id、文件名、日志 category、bundle 路径不是玩家可见文本，不纳入本清单。

## 当前结论

- 未发现未处理的玩家可见英文文案。
- 关键按钮具备基础读屏标签。
- 界面缩放入口可用，窗口缩放时地图世界层会重新布局。
