# 12 · 角色圣经（约 35 名花名册）

> 归属：[B · 叙事与导演](README.md) ← [设计蓝图](../README.md)
> 本页方法论：narrative-design（角色弧光：欲求/需要/缺陷/转变）+ GameDesignAssistant（反派动机与花名册角色多样性）。
## 一、角色设计框架（每名可招募角色适用）
| 维度 | 问题 |
| --- | --- |
| 欲求 Want | 角色表面追求什么（推动情节） |
| 需要 Need | 角色真正欠缺什么（弧光终点） |
| 缺陷 Flaw | 阻碍其成长的性格弱点 |
| 转变 Arc | 从何种状态到何种状态 |
| 机制逆映 | 人设如何体现为职业/成长/羁绊（见 [A/04]） |

## 二、双主角（完整档案）
**奥德里克（Aldric）**：兄，索雷因圣骑士。
- 欲求：保卫王国与秩序。需要：学会质疑不义的权威。缺陷：压抑、愁守教条。
- 转变：「秩序的信徒」→「质疑神明的反抗者」。初始职：枪骑 → 圣骑士 → 龙王/圣痕使。
**艾拉菈（Elara）**：妹，诺德海姆女武神/巫女。
- 欲求：为族人复仇、护卫自由。需要：学会为爱放下仇恨。缺陷：冲动、不信任。
- 转变：「复仇的战士」→「愿为爱牺牲者」。初始职：天马 → 天空骑士 → 龙王/圣痕使。

## 五、阵容角色多样性（role 覆盖）
| 战术角色 | 代表角色 |
| --- | --- |
| 前排肉盾 | 将军型、圣殿卫 |
| 机动刃 | 飞兵、骑兵、刺客 |
| 远程输出 | 弓手、法师、魔导器械 |

确保每种战术角色至少 3–4 名可选，避免玩家被迫“无人可用”。
> 📌 每名角色需含：立绘规格（见 [C/15]）、初始职业与成长（[A/04]）、支援配对（[B/13]）、登场章节（[B/11]）、欲求/需要/缺陷/转变。逐条产出追踪见 [F/23]。

## 六、人物弧线与反派原则
- 每个可招募角色至少一条**支援线**揭示其秘密或成长。
- **反派有可理解的动机**：塞西莉亚的忠诚被利用、宗座的“秩序”信念；反派不是为恶而恶。
- 双生的**禁忌情愛**处理为情感张力而非猎奇，服务“宿命重压”主题。
- 角色思想/口疗与阵营命名风格一致（见 [B/10]）。
