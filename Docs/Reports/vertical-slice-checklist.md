# 单区域竖切验收清单

## 验收路径

- 从主菜单点击“新游戏”。
- 在职业选择中任选 2 人，例如“战士 + 法师”或“弓箭手 + 刺客”。
- 进入“裂隙村外缘”地图后，点击地面移动队长，按 `Tab` 切换队长。
- 点击“和村长交谈”，接下任务“血债”。
- 走到地图中部偏右的遭遇区域，触发“受惊野猪”战斗。
- 战斗中先点击“移动”，再点场景地面移动当前角色；选择“劈砍 / 火花 / 瞄准射击”等技能后，再点击红色描边敌人作为目标。
- 法师的“火花”命中油污区域附近目标时会生成火焰地表反馈。
- 击败野猪与偷猎者后点击“返回探索”。
- 打开“任务日志”确认中文任务文本可见。
- 打开“背包”和“存档”，确认 UI 能进入并返回。

## 竖切内容

- NPC：村长。
- 任务：血债，单线接取与交付。
- 遭遇：受惊野猪 + 偷猎者，分别覆盖近战压迫与远程威胁。
- 地图：草地、土路、水边、木堆障碍、油污区域、石地边界。
- 地表：水、油；火花技能可生成火焰并点燃油污。
- 奖励：旧箱子位置记录在 `item` 对象层，奖励物品为锈剑。
- 美术：自制像素 tileset、角色、敌人、NPC、宝箱、木堆 PNG，资源清单记录 license。

## 已知边界

- P2 只验收单区域竖切，不代表首章完整内容量。
- 战斗移动使用连续距离和整数 AP；暂不做复杂 navmesh 战斗绕障路径。
- 消耗品入口已保留，但 P2 尚未配置战斗消耗品。
- P3 需要扩展到村庄、野外、洞穴 8-10 个区域和 4-5 场固定战斗。
