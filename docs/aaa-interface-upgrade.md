# AAA 风格界面与内容升级记录

本轮升级以“不推倒现有可玩骨架”为前提，保留 SwiftUI + SpriteKit + RiftCore 的架构边界，重点提高信息层级、战术可读性、叙事密度和操作反馈。

## 体验支柱

1. **裂隙调查感**：所有主界面围绕“档案、封层、证据、战术终端”建立统一视觉语言。
2. **双人小队辨识度**：职业拥有姓名、称号、职责、背景说明和独立色彩；编成顺序与当前行动者始终可见。
3. **战术清晰度**：生命、AP、行动顺序、目标提示、技能费用和敌我关系在战斗中同时通过文字、图标与颜色表达。
4. **剧情驱动系统**：任务不再只是标题与一句摘要，而是包含区域指引、调查步骤、证据链和更具体的角色动机。
5. **反馈分层**：移动点击、命中、受伤、技能效果与 UI 状态切换均增加即时视觉响应。

## 验收基线

- `RiftCore` 38 项单元测试全部通过。
- 所有 JSON 内容文件通过语法校验，并由 `ContentLoader` 在测试中完成真实解码。
- 全部 Swift 源文件通过 Swift 前端语法解析。
- 最终 macOS 视觉、字体、窗口缩放和 SpriteKit 特效仍应在 Xcode/macOS 14+ 上完成一次截图验收。

## 后续制作边界

本轮实现的是可运行的“AAA 风格表现层与内容增强”，不是以大型美术团队、动作捕捉、全语音和高规格 3D 资产为基础的真正 3A 制作规模。继续提升时，优先投入原创角色立绘、环境关键画、分层音效、动态音乐、更多战斗动画和章节内容，而不是继续堆叠 UI 装饰。
